主な機能

概要欄まわりの面倒、ぜんぶここで終わらせる

個別編集を 1 本ずつ繰り返してた作業を、選択 → ルール指定 → 適用の 3 ステップに圧縮します。

一括置換・追加・削除

選択した動画の概要欄やタイトルに対して、文字列置換・末尾追加・先頭追加をまとめて実行。タイトルにも対応。

テンプレート

定型のスポンサー文・SNS リンク・章立てなど、よく使う文章を保存して呼び出すだけ。使用回数で人気のテンプレが一目で分かります。

差分プレビュー

適用前に全動画の変更前 / 変更後を Before / After で全件確認。一致箇所は緑、削除箇所は赤でハイライト。

実行履歴 + ワンクリック復元

適用前に自動でスナップショット作成。30 日以内ならいつでも履歴から復元できるので、安心して試せる。

リンク切れ検出

全動画の概要欄をスキャンして 404 / 410 のリンクを検出。古いアフィリエイトや旧チャンネル URL の置換に。

公開設定の一括変更

選んだ動画の公開設定（公開 / 限定公開 / 非公開）をまとめて切り替え。古い動画の一括非公開化やリリース解禁時にも便利。

使い方

3 ステップで終わる

インストール後、Google アカウントで認証するだけ。データは全てブラウザ内で処理されます。

動画を絞り込む

期間・キーワード・「特定の文字列を含む」など複数条件で対象動画を絞ってから選択。

編集ルールを組む

置換 / 末尾追加 / 先頭追加のモードを選んで内容を入力。保存済みテンプレートから読み込みも可能。

差分を確認して適用

変更前 / 変更後を全件確認。「N 本に適用」を押すだけで自動バックアップ込みで実行。
