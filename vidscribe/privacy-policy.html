プライバシーポリシー

最終更新: 2026-01-05 対象: VidScribe Chrome 拡張機能

本ポリシーは、Chrome 拡張機能「VidScribe」（以下「本拡張」）が 利用者のデータをどのように扱うかを説明するものです。 本拡張は、利用者の個人情報を外部サーバーへ送信せず、すべての処理はブラウザ内で完結します。

取得するデータ

本拡張は以下のデータを取得・処理します：

  • 動画データ: YouTube API 経由で、利用者のチャンネルの動画タイトル・概要欄を取得します（編集機能の提供に必須）。
  • Google 認証トークン: API 認証のための OAuth トークン。Chrome の identity API で管理されます。
  • ローカル保存: 設定、テンプレート、実行履歴（スナップショット）、チャンネル情報のキャッシュをブラウザのローカルストレージ（chrome.storage.local）に保存します。
  • ライセンスキー: Pro 版を購入した場合、ライセンスキーはローカルに保存され、Gumroad の API で照合されます。

データの利用目的

取得したデータは以下の目的にのみ利用します：

  • 動画一覧の表示と選択 UI の提供
  • 利用者の指示に基づく動画タイトル・概要欄の更新
  • 設定・テンプレート・履歴の保存
  • Pro 版ライセンス状態の確認

外部送信について

本拡張は以下を一切行いません：

  • 利用者の個人情報の収集
  • 外部サーバーへのユーザーデータ保存
  • 第三者へのデータ共有・販売
  • 広告目的でのデータ利用
  • ブラウジング履歴のトラッキング

第三者サービス

本拡張は以下の外部サービスと通信します：

  • Google API: 動画データの取得・更新のため（適用規約: Google プライバシーポリシー）
  • Gumroad API: Pro 版ライセンスキーの照合のみ（適用規約: Gumroad プライバシーポリシー）
  • Google Fonts / fonts.gstatic.com: UI フォントの配信のみ

セキュリティ

すべてのデータ処理は利用者のブラウザ内で行われます。認証は Google の OAuth 2.0 プロトコルに基づき安全に処理され、当方のサーバーで機密情報を保管することはありません。

利用者の権利

利用者は以下の操作をいつでも行えます：

  • Chrome から本拡張を削除してアクセスを完全に取り消す
  • Chrome の拡張機能設定からローカル保存データを消去する
  • Google アカウントの権限ページで本拡張への API アクセスを取り消す

ポリシーの変更

本ポリシーは必要に応じて改定される場合があります。改定があった場合は本ページに反映し、上部の最終更新日を更新します。

お問い合わせ

本ポリシーに関するご質問は、Chrome Web Store の本拡張ページのサポート窓口までお寄せください。
